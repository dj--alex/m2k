Схема работы генератора билдов dj--alex/M2K

Исходный проект в nazwa.ini содержит имя игры
точно также называются папки с которыми ведёт работу скрипт создания билда

Все скрипты запускаются из основной папки проекта M2K 
rgenerate_public_UNIVERSAL.sh
rgenerate_DEB_universal.sh
rgenerate_android.sh

скрипт берет файлы из папки M2K  и обновляет билд игры в M2K-Linux
M2K-Linux/usr/bin содержит Levels i Saves используемые игрой для записи сохранений,настроек и новых уровней.
Уровни также могут читатся игрой как External level по требованию игрока. 

все релизы попадают в папку NEW автоматически.

rebuild.sh - лишь подготавливает файлы к новом билду и увеличивает его номер на 1. (build.ini)
в данном архиве все файлы уже готовы.
